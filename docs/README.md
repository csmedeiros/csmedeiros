# Caio Medeiros — AI Engineer Portfolio

Personal portfolio website built as a static site for GitHub Pages.

## Files

- `index.html` — main portfolio page
- `styles.css` — stylesheet (dark, modern, AI-themed)
- `profile.png` — profile photo

## Deploy to GitHub Pages

1. Create a new repository named `<your-username>.github.io` (or any repo).
2. Push these files to the `main` branch.
3. Go to **Settings → Pages**.
4. Under **Source**, select **Deploy from a branch**, choose `main` and `/ (root)`.
5. Save. Your portfolio will be live at `https://<your-username>.github.io/` (or `https://<your-username>.github.io/<repo>/`).

## Customization

Open `index.html` and update:

- `mailto:hello@example.com` — replace with your real email.
- LinkedIn URL: currently `https://www.linkedin.com/in/csmedeiros` — confirm or update.
- GitHub URL: currently `https://github.com/csmedeiros` — confirm or update.
